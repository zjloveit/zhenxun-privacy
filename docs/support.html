技術サポート · FAQ

「地震速報ヘルパー」のご利用に関するよくある質問とお問い合わせ方法をご案内します。

本アプリは気象庁の公式緊急地震速報アプリではありません。生命・安全に関わる判断は、必ず気象庁・自治体・テレビ・防災無線などの公式情報を優先してください。

よくある質問（FAQ）

地震情報が表示されない

  • インターネット接続（Wi‑Fi / モバイルデータ）を確認してください。
  • ホーム画面を下に引いて更新（プルリフレッシュ）してください。
  • データは第三者サービス Wolfx Open API 経由で取得しています。サービス側の障害・メンテナンス時は表示が遅延または停止する場合があります。

緊急地震速報の通知が来ない

  • iOS の「設定 → 通知 → 地震速報ヘルパー」で通知が許可されているか確認してください。
  • アプリ内「設定」で最小マグニチュード・最大通知距離のしきい値を確認してください。条件を満たさない地震は通知されません。
  • 「リアルタイム受信（WebSocket）」をオンにすると速報の受信が早まる場合があります（バッテリー消費が増えることがあります）。
  • 訓練報・取消報は通知対象外です。

位置情報・到達時間の推定について

  • 位置情報は任意です。許可すると震源からの距離と到達時間の推定に使用します。
  • 推定値は参考情報であり、実際の揺れの強さや到達時刻を保証するものではありません。
  • 位置情報は「設定 → プライバシーとセキュリティ → 位置情報サービス」でいつでもオフにできます。

広告が表示されない

  • 無料版では Google AdMob による広告を表示します。
  • 通信環境や広告在庫の都合で、一時的に広告が表示されない場合があります。
  • 緊急地震速報の全画面表示中は、安全のため広告を表示しません。

公式アプリとの違いは？

気象庁や自治体が提供する公式の防災アプリ・緊急地震速報と併用することを強くおすすめします。本アプリは補助的な情報ツールです。

お問い合わせ

不具合の報告・機能に関するご質問は、以下のいずれかからご連絡ください。

  • App Store：アプリの商品ページから「デベロッパに連絡」（開発者向けメッセージ）
  • GitHub：Issues（お問い合わせ） に投稿（日本語・中国語・英語可）

お問い合わせの際は、iOS のバージョン、端末機種、問題の再現手順をお書きいただけると対応がスムーズです。生命に関わる緊急事態の連絡先ではありません。
