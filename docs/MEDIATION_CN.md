# 个人开发者 · 中国大陆广告说明

## 穿山甲 / 优量汇 / 百度联盟

国内主流联盟（**穿山甲、优量汇、百度**等）通常要求 **企业或个体工商户** 资质，**个人身份证** 往往无法开通变现。这不是 App 代码能绕过的限制。

若你只有个人开发者账号，**不必**再集成穿山甲 SDK。

## 个人可用方案（推荐顺序）

### 1. 继续用 Google AdMob（当前工程已接好）

- [AdMob](https://admob.google.com/) 支持 **个人** 注册收款（按你所在地区政策，可能需税务信息）。
- 大陆用户 **填充率可能偏低**、出现空白横幅较常见，不等于配置错误。
- 上架后有真实流量后，填充通常会慢慢好转。

### 2. AdMob 中介里加「支持个人」的国际源（可选）

在 AdMob → **中介** 中可尝试添加（以 AdMob 后台当前列表为准）：

| 来源 | 说明 |
|------|------|
| **Unity Ads** | 个人可注册，有官方 AdMob 适配器 |
| **Meta Audience Network** | 需 Meta 开发者账号 |
| **AppLovin** 等 | 视地区与审核要求而定 |

这些仍走现有 `BannerAdView` + AdMob 广告位 ID，需在 Xcode 额外加对应 **Mediation Adapter** 包（与穿山甲类似，但注册门槛通常低于国内联盟）。

### 3. 以后若要国内高填充

- 注册 **个体工商户** 或 **公司**，再申请穿山甲 / 优量汇；或  
- 找有资质主体 **联运/授权**（注意合规与合同）。
