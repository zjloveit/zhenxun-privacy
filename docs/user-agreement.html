地震速報ヘルパー 利用規約

更新日：2026年6月1日

「地震速報ヘルパー」をご利用いただきありがとうございます。本規約に同意のうえご利用ください。

一、サービスの性質

  1. 本アプリは地震情報の閲覧、地図表示、第三者データに基づく通知および到達時間の推定を提供する情報ツールです。
  2. 本アプリは気象庁の公式緊急地震速報発表アプリではありません。
  3. 情報の遅延・誤差・欠落があり得ます。安全を保証するものではありません。

二、ユーザーの責任

  • 法令に反する利用をしないこと
  • 地震時は直ちに公式の避難情報に従うこと（本アプリのみに依存しないこと）

三、知的財産

アプリの UI・コードは開発者に帰属します。地震データの権利は各提供元に帰属します。

四、免責

地震、通信障害、データ誤り、端末故障等により生じた損害について、開発者は法令で認められる範囲を超えて責任を負いません。

五、広告

無料版には第三者広告（Google AdMob 等）が含まれます。緊急地震速報の全画面表示中は広告を表示しません。

六、変更・終了

本規約は変更される場合があります。いつでもアンインストールにより利用を終了できます。

七、準拠法

本規約は日本法を準拠法とします。紛争は開発者の所在地を管轄する裁判所を第一審の専属的合意管轄とします。
