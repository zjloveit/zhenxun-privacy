# 震讯 · 中国大陆 App Store 上架清单

## 应用定位（务必遵守）

| 项目 | 建议 |
|------|------|
| App 名称 | **震讯 - 地震速报与提醒** |
| 副标题 | 地震速报地图 · 第三方提醒 |
| 避免使用 | 「官方预警」「国家发布」「政府授权」等表述 |

## App Store Connect 必填

1. **隐私政策 URL**  
   将 `docs/隐私政策-网页版.md` 发布到您的网站（GitHub Pages、Notion 公开页等），填入 App Store Connect。

2. **App 隐私问卷**  
   - 定位：用于 App 功能，不用于追踪  
   - 设备 ID / 广告标识符：用于第三方广告（AdMob）  
   - 与第三方共享：是（广告、地震数据接口）

3. **出口合规**  
   已在 Info.plist 设置 `ITSAppUsesNonExemptEncryption = false`（仅使用 HTTPS）。

4. **年龄分级**  
   建议 4+，无受限内容。

5. **审核备注（建议原文粘贴）**

```
本应用为第三方地震信息服务工具，非中国地震台网中心或政府部门的官方地震预警发布应用。
地震提醒数据来自第三方开放 API，界面与通知均标注「非官方」。
首次启动需用户勾选同意隐私政策与用户协议。
预警全屏界面不展示广告；广告仅出现在首页、速报列表、设置页底部横幅。
```

## 中国大陆额外合规

| 事项 | 说明 |
|------|------|
| ICP 备案 | 若提供经营性互联网信息服务，需向省级通信管理局备案 |
| 算法/推荐 | 本应用无个性化推荐算法，一般无需算法备案 |
| 地震预警资质 | 法定地震预警发布需主管部门授权；本应用定位为信息工具可降低风险 |
| 数据来源 | 长期运营建议对接中国地震台网中心定制服务 |

## 推荐描述（可复制修改）

**震讯** 帮助您查看近期地震速报、地图分布，并在收到第三方来源的地震提醒时进行本地提示与到达时间估算。

**重要说明：** 本应用不是政府官方地震预警发布平台，数据可能存在延迟，不能替代政府权威发布。遇地震请立即按应急部门指引避险。
