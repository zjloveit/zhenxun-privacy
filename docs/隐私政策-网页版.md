「地震速報ヘルパー」（以下「本アプリ」）は、地震情報の表示とローカル通知を提供します。

## 一、収集する情報

1. **位置情報（任意）**：震源からの距離・到達時間の推定に使用（拒否可）
2. **デバイス・広告識別子**：広告・品質改善（IDFA は ATT 許可時）
3. **ネットワーク通信**：Wolfx API への地震データ取得（機微情報は送信しない）

## 二、利用目的

- 地震情報・地図・通知
- 第三者広告（Google AdMob）

## 三、第三者サービス

- 地震データ：Wolfx Open API（気象庁情報の転送、非公式）
- 広告：Google AdMob

## 四～九
